Organización y Procesamiento de Datos

Para organizar y procesar los datos meteorológicos es necesario asegurarse de que los archivos tienen un formato uniforme y que los datos sean consistentes y correctos.

Paso 1: Lectura de archivos

Se revisan las cabeceras, la separación entre datos y el tipo de delimitador (espacios, comas o tabulación). También se identifican las columnas y los tipos de datos.

Paso 2: Verificación de formato

Se comprueba que todos los archivos tienen el mismo formato mediante un script de validación que analiza las primeras filas de cada archivo.

Paso 3: Revisión de datos

Se asegura que los datos no contengan errores ni valores faltantes. Se usa pandas para gestionar los archivos y manejar errores de lectura, asegurando la consistencia de columnas.

Paso 4: Documentación

Se registra cada decisión tomada, cómo se han tratado los valores nulos y las soluciones aplicadas a las inconsistencias.

Análisis y Estadísticas

  • Cálculo del porcentaje de datos carentes (-999).
  • Medias y totales anuales de precipitación.
  • Tendencia de cambio en las precipitaciones.
  • Identificación de los años más lluviosos y más secos.
  • Otros análisis relevantes según los datos disponibles.
