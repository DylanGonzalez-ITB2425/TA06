Obtención de Datos

Para realizar el análisis climático, es fundamental obtener datos fiables de fuentes meteorológicas oficiales. En este caso, utilizaremos la AEMET (Agencia Estatal de Meteorología).

El proceso de obtención de datos sigue estos pasos:

  • Acceso a la fuente de datos: Ir al portal OpenData de AEMET y localizar los archivos de datos requeridos.
  • Solicitud de API-Key: Antes de poder descargar los datos, es necesario solicitar una API-Key a AEMET para obtener acceso.
  • Descarga de archivos: Obtener los archivos precip.MIROC5.RCP60.2006-2100.SDSM_REJ.tar.gz, que contienen la información climática necesaria para el análisis.

Estos datos serán procesados posteriormente para su análisis y visualización en forma de gráficos y estadísticas relevantes.
